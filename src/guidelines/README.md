## 📁 폴더 구조

```
guidelines/
├── README.md              # 이 파일
├── COMPONENT_GUIDE.md     # 컴포넌트 개발 가이드
├── API_GUIDE.md           # API 사용 가이드
```

## 🎯 목적

- **일관성**: 팀 전체가 동일한 패턴과 규칙을 따르도록 합니다.
- **효율성**: 새로운 개발자가 빠르게 프로젝트에 기여할 수 있도록 합니다.
- **유지보수성**: 코드베이스의 품질을 유지하고 향상시킵니다.
